## 이 문서의 역할

이 문서는 HSI가 완성된 표준지표라는 주장을 하기 위한 자료가 아니다.  
HSI가 왜 이러한 구조를 가지게 되었는지 설명하는 방법론 초안이다.

최종 성과와 결론은 `01_최종보고서`에, 계산식과 검증 로그는 `03_보충부록`에 두고,  
본 문서는 HSI가 형성되고 검증되는 과정을 보여주는 서사형 배경 자료로 읽는 것이 적절하다.

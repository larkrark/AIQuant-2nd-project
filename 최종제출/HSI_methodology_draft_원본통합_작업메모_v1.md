# HSI_methodology_draft 원본통합 작업메모 v1

## 작업 목적

사용자가 제공한 `HSI초기 아이디어와 구상의 출발점.hwpx` 원본을 기준으로, `HSI_methodology_draft.md`의 `## 5. HSI 방법론 정리: 발상에서 계산 구조로 줄어든 과정` 앞부분을 원본 기반 서사로 채우고, 5장 이후는 원본 문장을 중심으로 기존 HSI 방법론 정리본의 계산 구조와 검증 내용을 보완하였다.

## 반영 방식

1. 원본의 발상 배경을 최대한 보존하였다.
   - 6월 8일 반도체 충격 관찰
   - 시간축, 원점, 벡터
   - 물리학·지구과학·무기화학에서 온 사고방식
   - P파/S파, 지진계, 외부 경고와 내부 반응
   - 하인리히 법칙과 사건 강도 구상
   - 미니 프로젝트와 팀 프로젝트로 축소된 과정

2. 원본 문장의 의미는 유지하되, 보고서 문체에 맞게 정리하였다.
   - 금융시장에 자연과학 개념을 그대로 적용한 것처럼 보이는 표현은 완화하였다.
   - 구현하지 못한 아이디어는 배경 구상으로 명시하였다.
   - 최종 구현은 가격 기반 HSI 상태분류와 ETF 목표비중 연결로 축소되었다고 설명하였다.

3. 5장 이후는 기존 방법론 정리본과 앞선 수정사항을 반영하였다.
   - 입력 신호 제한 이유
   - 방향 통일
   - rolling z-score와 clipping
   - `V_plus`, `V_minus`
   - `m_v`, `risk_component`, `relief_component`
   - `direction`, `intensity`, `conflict`
   - HSI 5상태 판정
   - `THETA_COMMON + ACCIDENT_EXTRA`
   - ETF 목표비중과 λ 부분조정
   - 정보누수·과적합 검증 관점

4. 그림 자료를 보존하였다.
   - HWPX 내부 이미지를 `HSI_methodology_draft_figures/` 폴더로 추출하였다.
   - Markdown에는 상대경로로 연결하였다.
   - 이미지와 함께 보려면 zip 패키지를 사용하는 것이 좋다.

## 의도적으로 과하게 넣지 않은 내용

- 최종 성과표 전체
- Exp 44 극단값 override 전체 상세표
- 상용 로보어드바이저 사례 전체 각주
- 계산식 익힘장 수준의 숫자별 상세 예제

이 내용들은 각각 최종보고서, 업데이트 로그, 방법론 정리본, 계산식 익힘장에 두는 것이 더 적절하다.

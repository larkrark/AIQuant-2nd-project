# 01. 아침회의 안건

## 핵심 회의 안건

| 번호 | 안건 | 확인할 내용 | 상태 |
|---:|---|---|---|
| 1 | ETF 후보군 최종 확인 | 현재 예비 ETF 3종을 그대로 사용할지, 후보군을 확장할지 확인 | 논의 필요 |
| 2 | 방어형 자산군 분류 기준 | 위험자산, 안전자산, 방어자산, 현금성 자산 기준 확인 | 논의 필요 |
| 3 | HSI 입력 신호 목록 확정 | 수익률, 모멘텀, 이동평균 괴리, 변동성, 상대강도 등 입력 신호 확정 | 논의 필요 |
| 4 | 점수화 방식 합의 | 분위수 방식은 기본 실험 기준, z-score는 비교용 보조 실험으로 둘지 합의 | 합의 필요 |
| 5 | HSI 상태명 및 분류체계 정식화 | 현재 코드의 `buy / watch / caution` 라벨을 1차 계산 라벨로 유지하되, 보고서와 정식 1차 테스트에서는 위험 완화·관찰/중립·위험 악화·충돌 등 프로젝트용 HSI 상태명으로 매핑할지 확인 | 논의 필요 |
| 6 | 현재 예비 실행 결과 공유 | 일별 가격, 월말 가격, 월간 수익률, 분위수/z-score HSI 결과 저장 확인 | 공유 |
| 7 | 일별 HSI와 월별 백테스트 연결 방식 | 월말 HSI 상태를 추출해 다음 달 월간 수익률에 적용하는 방식 확인 | 합의 필요 |
| 8 | 기본 전략과 HSI overlay 연결 | 기본 전략 비중 위에 HSI를 어떻게 얹을지 확인 | 논의 필요 |
| 9 | 백테스트 기간과 리밸런싱 주기 | 월말 신호 → 다음 달 수익률 적용, 월간 리밸런싱 기준 확인 | 확인 필요 |
| 10 | 성과지표 기준 | CAGR, MDD, Sharpe, Turnover, 위기구간 성과 등 확정 | 논의 필요 |
| 11 | 발표용 그림과 표 구성 | 필수 그림, 선택 그림, 부록 그림 구분 | 논의 필요 |
| 12 | Grid Search / Robustness 위치 | 본 실험에 넣을지, 확장 검증으로 둘지 확인 | 합의 필요 |

## 회의에서 강조할 핵심 문장

> 현재는 전체 전략 성과를 확정한 단계가 아니라, HSI 입력 데이터 생성 및 분위수/z-score 신호 산출 예비 실행 결과를 확인한 단계이다.

> HSI 기본 실험은 분위수 방식으로 진행하고, z-score 방식은 삭제하지 않고 보조 비교 실험으로 유지한다.

> 현재 코드에서 사용되는 `buy / watch / caution` 라벨은 계산 결과를 빠르게 확인하고 중간 산출물을 점검하기 위한 1차 분류로 유지한다. 다만 보고서와 정식 1차 테스트에서는 이 라벨을 그대로 최종 상태명으로 사용하기보다, HSI의 원래 설계 의도에 맞게 별도의 상태 정의표를 두어 해석하는 방식이 적절하다.

> HSI는 단순히 매수, 관망, 주의와 같은 매매 판단을 바로 내리기 위한 신호라기보다, 가격 기반 시장환경신호를 위험 악화 방향과 위험 완화 방향으로 정렬한 뒤, 어느 쪽 신호가 우세한지, 신호의 강도는 어느 정도인지, 두 방향의 신호가 동시에 나타나는 충돌 상태가 있는지를 함께 해석하기 위해 설계한 체계이다.

> 따라서 `buy / watch / caution`은 1차 계산 라벨로 두고, 정식 실험과 보고서에서는 이를 위험 완화 우세, 관찰/중립, 위험 악화 우세, 필요 시 충돌 상태와 같은 프로젝트용 HSI 상태명으로 한 번 더 매핑한다.

> 현재 HSI 신호는 일별로 계산되어 있고, 백테스트에 사용할 수익률은 월별 수익률이다. 따라서 월말 HSI 상태를 추출한 뒤 다음 달 ETF 수익률에 적용하는 방식으로 시점 기준을 맞추는 것이 필요하다.

> 현재 Grid Search는 예비 점검용이며, 최종 Grid Search는 백테스트 성과지표인 CAGR, MDD, Sharpe, Turnover 등을 연결한 뒤 진행한다.

## 회의에서 조심할 표현

| 피할 표현 | 이유 | 대신 사용할 표현 |
|---|---|---|
| 전략 성과 검증 완료 | 아직 백테스트 연결 전이거나 정식 1차 테스트 전이므로 과장될 수 있음 | HSI 입력 데이터 생성 및 신호 산출 예비 실행 결과 |
| 전체 ETF 유니버스 결과 | 현재는 예비 ETF 3종 기준 | 예비 ETF 3종 기준 결과 |
| HSI 예측모델 | 미래 수익률 예측으로 오해될 수 있음 | 시장상태 판단용 보조지표 |
| `buy / watch / caution` 최종 투자신호 | 단순 매수·관망·주의 신호처럼 오해될 수 있음 | 계산 결과 확인용 1차 라벨, 보고서에서는 HSI 상태 정의표로 재해석 |
| z-score와 분위수 비교 프로젝트 | 프로젝트 주제가 바뀐 것처럼 보일 수 있음 | HSI 점수화 방식 보조 비교 실험 |

### HSI 기여도 분리 문제 제기 (김민호) — 핵심 논의
- 전략에 독립 정보원 2개: ①HSI가 정하는 목표비중(방향) ②변동성·rolling drawdown이 정하는 λ(속도)
- 위기 때 둘이 동시에 방어 방향으로 작동 → **성과 원인 분리 불가**. "HSI 없이 변동성만 봐도 되지 않느냐" 질문에 현 데이터로 답 불가
- 해법 제안: **HSI 목표비중을 무작위 셔플한 대조군** (λ 규칙은 유지)
- 김근형 보완: λ는 방향 신호가 아니라 이전 비중→목표비중 **이동속도** 개념임을 정정. 1회 셔플로는 부족 → 몬테카를로 다회 + 상태 지속성 보존을 위한 **블록(3~6개월) 셔플** 제안 → 합의
- 이미 37번 실험에서 월별 초과수익 = exposure effect + timing effect 분해 완료 (timing effect +25.80%p, FULL 기준)
- 권보성: HSI 방향 vs λ 기여 분리 ablation 시각화(MDD·Calmar 비교 차트) 제작 → 김민호 "HSI가 영향력 있다는 게 고무적"

### 38번 Shuffle Placebo Test 결과 (김근형, 저녁)
- 설계: 실현 λ_t 시퀀스 고정, HSI 상태 시퀀스를 블록 셔플 → 목표비중 매핑 → w_t = w_(t-1) + λ_t×(w*_shuffled − w_(t-1)) 재계산, N회 반복해 귀무분포 생성
- 결과(1,000회): **OOS Calmar 97.1%ile, MDD 95.2%ile, CAGR 93.7%ile, Sharpe 90.4%ile**. 단 IS CAGR 54.1%ile로 평범
- 해석: "HSI가 모든 구간에서 수익률을 잘 만든다"가 아니라 **"방어형 지표(MDD·Calmar)에서 강했다"**. 과적합 의심 회피를 위해 OOS 기준 권장
- 명칭 정정: λ 경로에 상태 조건 정보가 남을 수 있어 "HSI 독립 기여 확인"이라 말하면 안 됨 → **"HSI 목표비중 타이밍의 Shuffle Placebo Test"**로 표기

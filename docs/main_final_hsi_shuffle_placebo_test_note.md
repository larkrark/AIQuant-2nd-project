# HSI 목표비중 Shuffle Placebo Test 노트

## 방법론

λ 실행규칙(변동성·drawdown 기반 조절)은 dynamic_v1의 실현값을 그대로 고정하고, HSI 목표비중(상태 시퀀스)만 4개월 블록 단위로 1000회 무작위 셔플하여 대조군 포트폴리오를 생성하였다. 각 대조군의 CAGR, MDD, Sharpe, Calmar를 계산하여 귀무분포를 구성하고, 실제 dynamic_v1의 성과가 이 분포에서 위치하는 백분위를 확인하였다.

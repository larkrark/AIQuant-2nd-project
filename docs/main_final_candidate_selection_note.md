# 20~23번 Final Candidate Selection Note

## 목적
여러 실험 계열에서 나온 후보들을 거래비용, Turnover, MDD, Sharpe, Calmar, 비용 민감도 기준으로 다시 정렬하여 최종 보고서 후보를 압축한다.

## 핵심 결론
- 최종 후보: Lambda 0.1, Lambda 0.3
- 비교 기준: EW Benchmark
- 내부 기준선: HSI baseline
- 보조 진단: event balance, theta, signal combo, macro overlay
- Lambda 0.1은 저회전·보수형 후보
- Lambda 0.3은 수익성·Calmar 균형형 후보

## 주요 숫자
- Lambda 0.1: CAGR 8.623%, MDD -14.788%, Sharpe 0.791, Calmar 0.583, Turnover 2.515%
- Lambda 0.3: CAGR 8.995%, MDD -15.335%, Sharpe 0.775, Calmar 0.587, Turnover 6.950%
- EW: CAGR 6.510%, MDD -13.571%, Sharpe 0.832, Calmar 0.480, Turnover 0.000%

## 보고서 문장
최종 후보 선정 결과, Lambda 0.1과 Lambda 0.3은 HSI baseline 대비 Turnover와 MDD 부담을 크게 완화하면서 EW보다 높은 CAGR과 Calmar를 보였다. 다만 EW Benchmark의 Sharpe와 MDD가 강하므로, HSI-Lambda 후보가 모든 지표에서 절대적으로 우월하다고 단정하지 않고 투자 성향별 후보로 제시한다.

AIQuant 2차 프로젝트 One Page Proposal
조번호: 3조   |   조원: 권보성, 김근형, 김민호, 추주원
1. 프로젝트 제목
HSI(Hourglass Signal Index)를 활용한 미국 대형주 가격 시계열 분석 및 투자 신호 탐색
2. 프로젝트 내용
30개 미국 대형주의 1년 가격 데이터를 기반으로 수익률, 변동성, 추세 변화를 분석하고, 모래시계 구조의 HSI 지표로 종목별 위험·기회 신호를 시각화한다.
3. 최종 목표 및 결과물
종목별 수익률·변동성 EDA, HSI 산출 로직, 매수/관망/주의 신호 시각화, 분석 코드, 최종 발표 PPT를 완성한다.
4. 동기 및 배경
단순 가격 상승률만으로는 투자 판단이 어렵기 때문에, 추세의 방향성과 변동성 확대 구간을 함께 보여주는 직관적 신호 지표가 필요하다.
5. 프로젝트의 의미, 가치
주가 흐름을 정량 분석과 시각 모델로 동시에 설명하여 종목 비교, 위험 구간 탐지, 투자 의사결정 보조에 활용할 수 있다.
6. 차별성, 독창성
일반적인 수익률·변동성 분석에 그치지 않고, 가격 흐름을 모래시계 형태의 신호 구조로 변환하여 추세 수렴·확산과 방향성을 한눈에 표현한다.
8. 필요한 데이터 소스 및 확보 방법
현재 sample_30stocks.xlsx의 30개 종목 가격 데이터를 사용한다. 추가 확장 시 Yahoo Finance 등 공개 금융 API에서 OHLCV, 거래량, 지수 데이터를 확보한다.
9. 분석 내용 및 적용 기술
Python 기반 전처리, 일별 수익률·누적수익률·변동성 계산, 상관관계 분석, 이동평균/표준편차 기반 신호 생성, HSI 지표 설계 및 시각화(matplotlib/plotly 등)를 적용한다.
10. 프로젝트 수행 프로세스 및 일정
시장조사 및 주제 확정 -> 데이터 확인/전처리 -> EDA -> HSI 산식 설계 -> 종목별 신호 분석 -> 시각화 대시보드/차트 제작 -> 결과 해석 -> 발표자료 작성.
11. 인원 및 업무 분담
3조: 권보성, 김근형, 김민호, 추주원. 역할은 기획/데이터 전처리/EDA/HSI 모델링/시각화/발표자료 제작으로 분담한다.
12. 최종결과물 제출자료
One Page Proposal, 분석 데이터, Python 소스코드, EDA 및 HSI 결과 차트, 최종 발표 PPT 자료.
※ 현재 기획 단계 초안이며, HSI 산식과 역할 분담은 EDA 진행 후 구체화 예정

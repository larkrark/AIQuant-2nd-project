# HSI 재현용 팀 공유 안내

## 1. 폴더 목적

이 폴더는 단순 결과 공유용이 아니라, 팀원 누구나 새 한국 ETF 데이터를 넣고 같은 순서로 실행하여 HSI 결과를 다시 만들 수 있도록 정리한 재현용 작업 폴더이다.

HSI는 완성된 투자전략이나 표준 금융지표가 아니라, 한국 상장 ETF 가격데이터에서 계산한 여러 가격 기반 신호를 이용해 시장 상태를 해석하기 위한 탐색적 보조지표이다.

---

## 2. 사용 데이터

현재 분석에 사용한 데이터는 한국 상장 ETF 가격데이터이다.

* 원천 사용 파일: `data/raw/korea_etf.csv`
* 전처리 후 분석 파일: `data/processed/korea_etf_price_clean.csv`
* 사건 달력 파일: `data/reference/event_calendar_us_kr.csv`
* 분석 대상: 한국 상장 ETF 가격 시계열
* 분석 단위: 일별 가격데이터를 바탕으로 수익률을 계산한 뒤, 월별 기준으로 HSI 상태를 분류

원천 제공처가 KRX, FinanceDataReader, 강의 제공자료, 팀 내 수집자료 중 어디인지 최종 보고서에서 확인할 필요가 있다.

---

## 3. 새 데이터로 다시 작업하는 방법

새 ETF 가격데이터를 사용할 경우 다음 순서로 작업한다.

1. 새 데이터를 `data/raw/` 폴더에 넣는다.
2. 파일명이 바뀌면 `src/00_check_korea_etf_data.py` 안의 원천 데이터 경로를 수정한다.
3. 기존 `output/` 결과를 그대로 믿지 말고, 00번부터 08번까지 순서대로 다시 실행한다.
4. 새 결과표와 그림을 확인한다.

새 데이터는 최소한 다음 정보를 포함해야 한다.

* 날짜
* 종목코드 또는 티커
* 종목명 또는 ETF명
* 가격

데이터 형식이 다르면 `00_check_korea_etf_data.py`에서 컬럼명을 맞추는 전처리가 필요하다.

---

## 4. US 사건을 포함한 이유

본 분석의 실제 수익률 계산 대상은 한국 상장 ETF이다.

다만 코로나 급락, 글로벌 금리 인상, 기술주 충격과 같은 글로벌 사건은 한국 ETF 시장에도 영향을 줄 수 있으므로 사건 달력에 포함하였다. 더하여 이러한 글로벌 사건 구간에서 한국 ETF 자산군이 어떤 거동 특징을 보였는지 살피기 위해 US 사건도 기준 이벤트로 사용하였다.

따라서 US/KR 구분은 사건 발생 배경을 나타내는 라벨이며, 실제 HSI 계산은 한국 ETF 가격데이터를 기준으로 수행하였다.

## 6. 전체 실행 순서

프로젝트 루트 폴더에서 아래 순서대로 실행한다.

```powershell
python src\00_check_korea_etf_data.py
python src\04_event_count_index.py
python src\05_hsi_state_classifier.py
python src\06_visualize_event_waves.py
python src\07_split_design_validation_periods.py
python src\08_event_hsi_interpretation_summary.py
```

주의할 점은 Python 파일을 직접 실행하지 않고 반드시 앞에 `python`을 붙인다는 것이다.

예를 들어 아래 방식은 잘못된 실행이다.

```powershell
src\05_hsi_state_classifier.py
```

아래처럼 실행해야 한다.

```powershell
python src\05_hsi_state_classifier.py
```

## 8. HSI 상태명

현재 HSI 상태명은 다음과 같다.

* 강한 위험악화
* 위험악화
* 고변동성 혼합구간
* 불안정 과열
* 과열 후보
* 중립/혼조
* 회복 후보
* 안정적 회복 후보

고변동성 혼합구간은 큰 상승과 큰 하락이 같은 월에 함께 나타나고, 단기 변동성이 중기 변동성보다 높아지는 상태를 의미한다.

---

## 9. 기간 분리 기준

미래정보 사용 가능성을 줄이기 위해 결과를 설계용 기간과 검증용 기간으로 나누어 확인하였다.

* 설계용 기간: 2014-03 ~ 2022-12
* 검증용 기간: 2023-01 ~ 2026-06

현재 결과는 완전한 out-of-sample 검증이라기보다, 탐색적 HSI 기준의 기간별 안정성 확인으로 해석하는 것이 안전하다.

## 12. 주요 해석 요약

사건별 HSI 해석 요약표 기준으로 보면 다음과 같다.

* 코로나 급락 구간: 강한 위험악화가 주된 상태로 나타남
* 코로나 이후 유동성 반등 구간: 과열 후보와 강한 위험악화가 함께 높게 나타남
* 인플레이션 및 금리 인상 충격 구간: 강한 위험악화가 가장 많이 나타남
* 2차전지 테마 과열 구간: 한국 ETF 전체 기준에서는 위험악화와 과열 후보가 함께 나타남
* 2024년 8월 글로벌 기술주 및 엔캐리 청산 충격 구간: 강한 위험악화와 고변동성 혼합구간이 높게 나타남

## 13. HSI 활용 예시

HSI 상태명은 향후 자산비중 조절 규칙으로 확장할 수 있다.

예를 들어 강한 위험악화에서는 위험자산 비중을 줄이고, 회복 후보에서는 위험자산 비중을 늘리는 방식이다. 다만 현재 단계에서는 실제 투자비중 결정 공식으로 확정하지 않고, 시장상태 해석용 보조지표로만 사용한다.

---

## 14. 해석상 주의사항

HSI는 표준화된 금융 이론이 아니라 이번 프로젝트에서 가격 기반 신호를 종합하기 위해 만든 실험적 보조지표이다.

따라서 다음 표현은 피하는 것이 좋다.

* HSI가 위험을 예측했다
* HSI로 위기를 사전에 포착했다
* 성과가 검증되었다
* 투자전략으로 입증되었다

대신 다음 표현을 사용하는 것이 안전하다.

* 사건 구간에서 HSI가 강한 위험악화 상태를 보였다
* 사건 구간별 시장 상태를 해석하는 보조지표로 활용 가능성을 확인했다
* 탐색적 기준의 기간별 안정성을 확인했다
* 향후 자산비중 조절 전략으로 확장 가능하다

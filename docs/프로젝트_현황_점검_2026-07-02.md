### 2.2 현재 위치 (회의록·설계노트 기준)

- 6/29 회의록·`main_v3_experiment_design_note`에 따르면 **main_v2/v2b 기준선과 본문 시각화는 완료**, 이후는 신호조합 실험 → 제한 Grid Search → Turnover 필터 → 거래비용 민감도 → Robustness 순서로 후보를 좁히는 **검증 단계**로 넘어가는 지점입니다.
- 거래비용 가정은 확정됨: 비용률 `0.00% / 0.05% / 0.10% / 0.20%`, `월별 거래비용 = 월별 Turnover × 거래비용률`, 중심 시나리오 `0.10%`.
- Turnover 필터 예비 기준: `avg_turnover ≤ 0.05`, `max_turnover ≤ 0.25`.

## 3. 발표 전 잔여 작업 (우선순위)

1. **브랜치 통합/정리** — `parameterized_code_기초실험` 미push 커밋 반영, `main`·`codex` 변경사항을 발표용 단일 브랜치로 병합.
2. **데이터 품질 점검 마무리** — `check_missing_values()` 연도별 집계 버그 수정, 체크리스트 B 항목 완료.
3. **HSI 입력신호 정의표** — `signal_definition_table.csv`, `signal_direction_table.csv` 생성 (체크리스트 C).
4. **main_v3 실험 실행** — 설계표(`main_v3_*`)를 실제 신호조합 실험 + 제한 Grid Search + 거래비용 민감도 + Robustness 결과로 산출.
5. **비교전략 확장** — EW+HSI 외 Risk Parity/GMV/MDP + HSI overlay 백테스트 (체크리스트 J).
6. **유니버스 확정** — 예비 3종 → 최종 ETF 후보군 확정 후 재실행 여부 결정.
7. **최종 발표물** — 최종 PPT, 발표 스크립트, Q&A 답변지, 성과 요약표 정리 (체크리스트 15·16장).
